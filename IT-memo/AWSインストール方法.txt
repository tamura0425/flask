インストール

各種をインストールするにはスーパーユーザーになる必要がある

$ sudo su


MySQLのインストール
## yum install -y mysql-server

追加プログラム(PHPのMySQLネイティブドライバ)のインストール
## yum install -y php-mysqlnd


PHPをインストール
## yum install -y php



Apacheの再起動
## service httpd restart
